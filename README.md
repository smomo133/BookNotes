# BookNotes

안드로이드 어플리케이션을 2017년 공식 언어로 추가된 코틀린으로 
RxJava, 안드로이드 아키텍처 컴포넌트, Dagger 라이브러리 등을 적용해서 
개발한 토이 프로젝트입니다.

도서를 관리해 주는 앱으로 도서 바코드 인식 및 온라인 검색으로 책을 추가하고
간단한 서평이나 별점 및 서재 내 검색이 가능합니다.
